CSS Grid se torna a ferramenta favorita dos desenvolvedores

Uma pesquisa recente mostra que criar layouts virou uma tarefa muito mais simples após a popularização do Grid. Ele surgiu de uma proposta da Microsoft em 2012, buscando criar layouts web robustos e bidimensionais (linhas e colunas). Implementado inicialmente no Internet Explorer 10, foi padronizado e adotado por todos os principais navegadores entre 2016 e 2017 para resolver a complexidade de criar layouts responsivos. Algumas vantagens do CSS Grid é sua simplicidade na definição de layouts bidimensionais e a capacidade de criar designs mais flexíveis e responsivos.
